# Sprint 01 - DSL Parser

Implement a minimal parser for a simplified OpenSCENARIO-like DSL.

## Requirements

Support syntax:

* scenario declaration
* actor instantiation
* do serial():
* parallel():
* action invocation
* with: block

## Output

* AST (Python objects or JSON)
* Print AST for debugging

## Constraints

* Do NOT implement semantics yet
* Focus only on syntax parsing

## Deliverables

* parser.py
* ast_nodes.py
* example parsing result
